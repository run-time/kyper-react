# Introduction

Kyper is the MX Design System for application styles and components.

![introduction preview](/assets/introduction.png)

## Design System Goals

1. Create a shared language between all designers and developers
1. Act as a single point of truth for UI styles and assets
1. Act as a resource for front-end developers to preview components visually and view code examples.
1. Document components
    - When to use and not use components
    - How to implement components
    - Available modifiers for components
