# Text area

Text areas enable the user to interact with and input content and data. This component can be used for long form entries.

![textarea preview](/textarea/doc/assets/textarea-preview.png)

## Overview
Text areas are taller than text inputs and wrap overflow text onto a new line. They are a fixed height and scroll vertically when the cursor reaches the bottom of the field. The large initial size indicates that longer responses are possible and encouraged.

<video src="/textarea/doc/assets/textarea-scroll.mp4" type="video/mp4" controls autoPlay loop mute="true"></video>

## Height & aspect ratio
The height of the textarea is up to the designer to determine. It should align to our 8px grid. It is recommended to have the textarea be an aspect ratio of **2.3:1**. An easy way to calculate textarea height is to divide the width of your textarea by 2.3. 

![textarea aspect ratio](/textarea/doc/assets/textarea-ratio.png)
