# ButtonGroup
ButtonGroup is used to properly space a group of related buttons such as **Save** and **Cancel**

[Live demo](/?path=/story/components-buttongroup-examples--button-group-default)

## Overview

#### When to use

Use to group Button objects

#### When not to use

Do not use to group anything other than related buttons (differnt actions for the same target)
